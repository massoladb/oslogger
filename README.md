
# OS Logger

## Configuração do Ambiente

Para rodar este projeto, siga os passos abaixo no terminal (cmd):

### 1. Criar ambiente virtual
```cmd
python -m venv venv
```

### 2. Ativar o ambiente virtual
```cmd
venv\Scripts\activate
```

### 3. Instalar dependências
```cmd
pip install -r requirements.txt
```

Após concluir esses passos, o ambiente estará pronto para executar o código.

## Requisitos

- Python 3.11
